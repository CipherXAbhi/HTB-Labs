# About Tier 0
> This tier focuses on the essential building blocks of penetration testing. The main goal is to introduce users to the basics of connecting to Hack The Box, performing initial enumeration, and interacting with simple services. Each machine in this tier presents a single, straightforward step to obtain a flag, with plenty of guidance and walkthroughs available to help users build confidence and familiarity with the process.
---
# Skills you’ll gain
* Connecting to a VPN and accessing remote machines.
* Basic port scanning and service enumeration (e.g., using nmap).
* Interacting with common network services (SSH, HTTP, FTP, etc.).
